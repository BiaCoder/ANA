# 🚀 Gerenciador de Tarefas Pessoais

Um aplicativo de lista de tarefas (To-Do List) moderno, responsivo e focado na organização da rotina diária.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica da página.
* **CSS3**: Estilização personalizada, layout responsivo e efeitos visuais modernos.
* **JavaScript (ES6+)**: Manipulação do DOM para interatividade (adicionar, concluir e deletar tarefas).
* **Local Storage**: Persistência de dados local para evitar a perda de informações ao recarregar a página.

## 💻 Funcionalidades
* Adição dinâmica de novas tarefas (via clique no botão ou tecla Enter).
* Marcação visual de tarefas concluídas.
* Remoção individual de itens.
* Salvamento automático do estado da lista no navegador do usuário.
